# Usage with React SDK

## Quick Started

The quickest way to get started is to use the interactive [ConnectWallet playground ](https://thirdweb.com/dashboard/wallets/connect).

It allows you to customize ConnectWallet component and preview the changes in real time, You can then copy the code snippet and add it directly to your app.

## Get Started

Refer to [Getting Started with React SDK](/react/getting-started) and set up ThirdwebProvider in your project.

You can find the list of all wallets for React SDK [here](/react/connecting-wallets)

You can connect to a wallet in two ways:

1. using [ConnectWallet](/react/react.connectwallet) component for prebuilt UI
2. using [useConnect](/react/react.useconnect) hook to build a custom UI

## 1. Using ConnectWallet

ConnectWallet component renders a button which when clicked opens a modal to allow users to connect to wallets specified in the [ThirdwebProvider's supportedWallets](http://localhost:3000/react/react.thirdwebprovider#supportedWallets) prop.

#### Example

```tsx
import {
  ThirdwebProvider,
  metamaskWallet,
  coinbaseWallet,
  walletConnect,
} from "@thirdweb-dev/react";
import { ConnectWallet } from "@thirdweb-dev/react";

// wrap your app with ThirdwebProvider and pass supported wallets

function MyApp() {
  return (
    <ThirdwebProvider
      supportedWallets={[
        metamaskWallet(),
        coinbaseWallet(),
        walletConnect({
          projectId: "YOUR_PROJECT_ID",
        }),
      ]}
      clientId="your-client-id"
    >
      <App />
    </ThirdwebProvider>
  );
}

// render ConnectWallet in your app

function App() {
  return (
    <div>
      <ConnectWallet />
    </div>
  );
}
```

## 2. Using useConnect hook

[useConnect](/react/react.useconnect) hook allows you to programmatically connect to the wallet which allows you to create a custom UI for connecting to the wallet.

```tsx
import { useConnect, metamaskWallet } from "@thirdweb-dev/react";

const metamask = metamaskWallet();

function App() {
  const connect = useConnect();

  return (
    <button
      onClick={async () => {
        const wallet = await connect(metamask);
        console.log("connected to ", wallet);
      }}
    >
      Connect to MetaMask
    </button>
  );
}
```
